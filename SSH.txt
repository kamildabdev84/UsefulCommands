-----------------------------------------------------------------
Connect via SSH using other port
-----------------------------------------------------------------

ssh -p 1234 user@address.com

-----------------------------------------------------------------
Connect via SSH in debug mode
-----------------------------------------------------------------

ssh user@address.com -v

-----------------------------------------------------------------
Connect via SSH through proxyuser host to targetuser host
-----------------------------------------------------------------

ssh -o ProxyCommand='ssh -W %h:%p proxyuser@192.168.1.20' targetuser@10.0.0.1
